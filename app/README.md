# Zenkash

Personal finance management app built with Vue 3, Quasar, and Capacitor.

## Install dependencies

```bash
yarn install
```

## Development

```bash
# Start development server (hot-reload)
yarn dev

# Build for production
yarn build
```

## Code Quality

```bash
# Run ESLint and TypeScript checks
yarn lint

# Fix formatting with Prettier
yarn lintfix

# TypeScript type checking only
yarn typecheck
```

## Mobile

```bash
# Build Android APK (includes icons generation, version bump prompt)
yarn android

# Build for iOS
yarn ios

# Generate app icons from SVG
yarn assets
```
